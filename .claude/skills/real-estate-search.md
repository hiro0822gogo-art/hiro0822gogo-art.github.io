# real-estate-search スキル

## 保存済み検索条件

| 項目 | 条件 |
|---|---|
| **用途** | 障害福祉サービス（グループホーム / 生活訓練） または 自立準備ホーム |
| **エリア** | 滋賀県大津市周辺 |
| **建物構造** | 新耐震基準（1981年以降の建物） |
| **間取り** | 5LDK以上（なるべく大きめ） |
| **駐車場** | 2台以上（好ましい） |
| **利用形態** | 店舗利用可 / 事務所利用可 / 法人契約歓迎 |
| **補足** | 上記条件に合う物件がなければ、自立準備ホームとして使えそうな物件も候補に含める |

## 手順

1. 以下のサイトを WebSearch で検索する：
   - SUUMO：`site:suumo.jp 大津市 5LDK 店舗兼用 法人`
   - HOME'S：`site:homes.co.jp 大津市 5LDK 事務所利用可`
   - AtHome：`site:athome.co.jp 大津市 5LDK 法人`
   - 一般検索：`大津市 5LDK 新耐震 店舗利用可 グループホーム 賃貸`

2. 見つかった物件を以下のフォーマットで整理する：

```
## 候補物件リスト

### 物件A
- 住所：
- 間取り：
- 築年数（1981年以降か）：
- 駐車場：
- 賃料/売値：
- 店舗・法人利用：可 / 要相談
- URL：
- メモ：
```

3. グループホーム認可に関わるチェックポイントも合わせて確認する：
   - 居室が個室で4.5畳以上あるか（障害者グループホームの設備基準）
   - 共有スペース（居間・浴室・トイレ）が確保できるか
   - 消防法上の用途変更が必要かどうか

## 関連法令メモ

- **障害者グループホーム（共同生活援助）**：障害者総合支援法に基づく。都道府県・市の指定が必要。建物は消防法・建築基準法の基準を満たす必要あり。
- **生活訓練**：同じく障害者総合支援法。通所・宿泊型あり。
- **自立準備ホーム**：法務省委託事業。特段の建物基準はないが、生活環境として適切であることが求められる。
